01 - Jumping Right In

This chapter is the quick-start Angular todo app. It shows the basic shape of an Angular project before getting deep into theory.

What is going on

The code lives in 01 - Jumping Right In/todo. It is intended to get a working app on screen quickly: components render data, templates show the UI, and Angular updates the page as state changes.

Use this chapter to see the minimum moving parts: project structure, app component, template binding, and running the development server.

What to look for

  • The root Angular component.
  • The todo data model.
  • Template syntax that loops through or displays values.
  • How the project is bootstrapped.
